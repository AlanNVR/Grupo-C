# Evidencias del proyecto SIMPA

Esta carpeta contiene las evidencias del proceso de modelado, configuración, generación, compilación y verificación de código del proyecto **SIMPA: Sistema Inteligente de Mantenimiento de Palma Africana**.

## Organización

| Evidencias | Contenido |
|---|---|
| EV01–EV04 | Desarrollo del diagrama de clases. |
| EV05–EV08 | Perfil UML y estereotipo `Entity`. |
| EV09–EV10 | Validación del modelo. |
| EV11–EV19 | Configuración y generación de código Java. |
| EV20–EV23 | Compilación y unidad demostrable. |
| EV24–EV26 | Implementación y ejecución del CRUD de `Lote`. |

Las evidencias deben conservar su numeración y mostrar claramente el proceso o resultado documentado.
